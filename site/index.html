Talk to Your Docs

A documentation Q&A agent that connects to hosted MCP docs servers, searches source material, and turns scattered pages into useful answers.

View repository Run the app

What it does

Documentation becomes conversational

AI

AI answers

Uses Nebius AI with DeepSeek-V3-0324 to answer technical questions clearly.

MCP

Docs-native search

Connects to MCP search endpoints so answers stay grounded in real documentation.

URL

Flexible sources

Point it at Mintlify-style docs or any compatible streamable HTTP MCP server.

UI

Streamlit chat

Provides a clean chat workflow with example prompts and sidebar configuration.

Architecture

A focused agent loop

The Streamlit UI collects the prompt, Agno orchestrates the agent, MCPTools retrieves relevant docs, and Nebius generates the final answer.
